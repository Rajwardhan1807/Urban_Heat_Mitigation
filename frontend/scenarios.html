Methodology

How It Works

Each scenario modifies specific input features (NDVI, albedo, NDWI) for eligible grid cells, then runs the trained XGBoost model to predict the new LST. The difference (ΔT) shows the estimated cooling effect. Only physically plausible perturbations are applied based on the PRD's intervention parameterization table.
